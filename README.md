# Sales Data Analysis

## Objective
To analyze sales performance and generate business insights related to revenue, product trends, and regional growth using public datasets.

## Tools & Skills
- Excel (data cleaning, pivot tables)
- SQL (aggregations, joins)
- Python (exploratory data analysis)
- Power BI (interactive dashboards)

## Dataset
Public sales dataset sourced from Kaggle.

## Key Analysis
- Revenue trends over time
- Top-performing products and regions
- Monthly and yearly growth patterns

## Outcome
This project helps business stakeholders understand sales performance and make data-driven decisions.

## Status
Initial version created using public datasets. Dashboards and Python notebooks will be enhanced further.
